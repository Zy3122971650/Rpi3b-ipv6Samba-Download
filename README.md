# 这是啥？
用Samba做文件共享，you-get下载音视频，aria2做普遍意义的下载的离线下载器、文件分享机。

## 想做什么？
把树莓派做成一个下载机和私人文件共享机，供大学舍友下载、储存、分享文件。
